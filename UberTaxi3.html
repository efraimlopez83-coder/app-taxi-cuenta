⚙️ Configurar Firebase

Para que funcione en varios móviles, crea un proyecto en Firebase y pega aquí la configuración. Después pulsa guardar y recarga la página.
En Firestore tendrás que crear una base de datos y poner reglas de prueba para empezar:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /pedidos/{document=**} { allow read, write: if true; } } }
